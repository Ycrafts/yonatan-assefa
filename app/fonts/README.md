# Fonts Directory

This directory should contain the following font files:

## Clash Display (Display/Heading Font)
- ClashDisplay-Medium.woff2
- ClashDisplay-Semibold.woff2
- ClashDisplay-Bold.woff2

## Satoshi (Body Font)
- Satoshi-Regular.woff2
- Satoshi-Medium.woff2
- Satoshi-Bold.woff2

## JetBrains Mono (Monospace Font)
- JetBrainsMono-Regular.woff2

## Where to get these fonts:

### Clash Display
Download from: https://www.fontshare.com/fonts/clash-display
- Free for personal and commercial use

### Satoshi
Download from: https://www.fontshare.com/fonts/satoshi
- Free for personal and commercial use

### JetBrains Mono
Download from: https://www.jetbrains.com/lp/mono/
- Free and open source

## Installation:
1. Download the fonts from the links above
2. Extract the .woff2 files
3. Place them in this directory (portfolio/app/fonts/)
4. The fonts will be automatically loaded by Next.js

## Temporary Fallback:
Until you add the font files, the site will use system fonts as fallbacks:
- Clash Display → sans-serif
- Satoshi → 'Plus Jakarta Sans', sans-serif
- JetBrains Mono → 'JetBrains Mono', monospace
